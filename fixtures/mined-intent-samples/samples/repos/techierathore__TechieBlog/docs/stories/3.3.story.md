# Story 3.3: Category Management

## Status

Completed ✅

## Story

**As an** author,
**I want** to organize posts into categories,
**so that** readers can find related content.

## Acceptance Criteria

1. Category CRUD operations implemented (Create, Read, Update, Delete)
2. Categories have name, slug, and optional description
3. Category selector dropdown in post editor
4. Post can belong to one primary category
5. Category archive page displays posts filtered by category
6. Admin can manage all categories from dedicated page
7. Categories displayed in sidebar for navigation

### FluentSelect Pattern for Category Dropdown

```razor
@inject CategorySvc CategoryService

<FluentSelect Label="Category"
              @bind-Value="selectedCategoryId"
              TOption="Category"
              Items="@categories"
              OptionText="@(c => c.CategoryName)"
              OptionValue="@(c => c.CategoryId.ToString())" />

@code {
    private List<Category> categories = new();
    private string selectedCategoryId = string.Empty;

    protected override async Task OnInitializedAsync()
    {
        categories = CategoryService.GetAllCategories().ToList();
    }
}
```
